### js환경 만들기
이제 로드할 js폴더를 만들고 그 안에 app.js, math.js를 만들어 준다

{% highlight js %}
//foldertree
webpack_work 
   |- __resource
       |- js
         |-app.js
         |-math.js
   |- node_modules
       |- package.json
       |- webpack.config.js
{% endhighlight %}

그리고 webpack.config.js를 수정해준다.
{% highlight js %}
const path = require('path');

module.exports = {
    mode: 'development',
    entry: {
        main: './js/app.js',
    },
    output: {
        path: path.resolve('../dist'), 
//const path = require('path') 모듈 사용해 절대경로 가지고옴
        filename: '[name].js', 
    }
};
{% endhighlight %}

잘 불러들이는지 확인용으로 js를 만들어 sum함수를 불려들여 실행해 보자

{% highlight js %}
//math.js
export function sum(a,b) {
    return a + b;
}

//app.js
import {sum} from './math.js';

window.addEventListener('DOMContentLoaded', () => {
    const el = document.querySelector('#app');
    el.innerHTML = `
        <h1>1+2 = ${sum(1,2)}</h1> 
    `;
});
{% endhighlight %}

잘 합쳐져 main.js로 만들어지는지 확인을 위해 build 명령어 실행한다

{% highlight js %}
__resource> $ npm run build
{% endhighlight %}

그러면 새로 생긴 dist폴더에 담긴 main.js가 보인다

{% highlight js %}
//foldertree
webpack_work 
   |- __resource
       |- js
         |-app.js
         |-math.js
   |- node_modules
       |- package.json
       |- webpack.config.js
   |-dist
       |-main.js
{% endhighlight %}

화면에서도 잘 보이는지 index.html를 만들어 로드 해보자

{% highlight js %}
//foldertree
webpack_work 
   |- __resource
       |- js
         |-app.js
         |-math.js
   |- node_modules
       |- package.json
       |- webpack.config.js
   |-dist
       |-main.js
   |-index.html
{% endhighlight %}

index.html에 들어갈 내용
{% highlight js %}
<!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
    <div id="app"></div>
    <script src="./dist/main.js"></script>
</body>
</html>

{% endhighlight %}

브라우저 여는 명령어를 사용한다

{% highlight js %}
__resource> $ npm run build
{% endhighlight %}

사용시 화면에서는 math + app js를 결합한main.js가 화면에 잘 나온다<br>
